## Harpoon

* Add file to quick toggle A
* Toggle between quick toggle files `CTRL + E`


## LSP CONFIG

Rename variable under cursor:  `grn`
Execute a code action (error or suggestion):  `gra`
Find references of word:  `grr`
Goto definition:  `grd`
Goto implementation:  `gri`

Fuzzy find in document:  `g0`

Toggle LSP hints:  Leader + `th`

Better python lsp?  pyright or python-lsp-server
Using two results in double prompting renaming (unless vim config is modified)
