# Awesome window manager

## Theming

Default themes stored in `/usr/share/awesome/themes`
* default: gray/black 
* gtk: orange 
* sky: light blue; 
* xresources: purple black
* zenburn:  gray/green


Some other themes:
https://github.com/lcpz/awesome-copycats






## Shortcuts
Some useful, default keyboard shortcuts; can be almost arbitrarily re-bound
* Alt j/K:  Toggle focus between windows on a workspace (screen)
* CTRL + Super + Space:  Toggle window floating mode
* Alt + Super + j/k:  Change horizontal spacing of windows
* Shift + Super + j/k:  Move window to previous/next index

## "Solved" issues

* Screenshot shortcut (PrtSc key):  flameshot but also tried gnome tool and scrot
* Screen locking: i3lock
* Easily managing WIFI:  nm-applet
* Easy bluetooth management:  blueman-applet
* Increase volume above "100%":  pasystray (pulse audio system tray utility)
* Control screen brightness:  fixed using "light" package
