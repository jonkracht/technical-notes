# PopOS 22.04 notes

## (Default) keyboard shortcuts

| Shortcut              | Behavior                          |
| ---                   | ---                               |
| `Super + /`           | Launcher                          |
| `Super + A`           | Launch applications menu          |
| `Super + M`           | Maximize window                   |
| `Super + SHIFT + L/R` | Move window to another monitor    |
| `Super + ENTER`       | Enter into window adjustment mode |
|                       |                                   |
