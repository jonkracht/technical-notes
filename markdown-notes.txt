# Text styling

*Italicized text*       (single asterisk)
**Bold text**           (double asterisk)
~~Strikethrough text~~  (double tilde)


# Horizontal line across page width  
Three hyphens (---)


# Lists

## Ordered
1.  Text
2.  More text
3.  Etc


## Unordered
Use hyphen, minus sign or plus sign


# Line breaks

## Single-spaced break  
End line with two spaces or <br/>


# Code segments

## Inline
Surround with back ticks
Ex. `sudo apt update` updates your repo lists

## Fenced block
```[LANG] 
SOME CODE
MORE CODE
```
where [LANG] provides syntax highlighting and can be Python, C, HTML, vim etc



# Links
[link text](www.link.url)




# Images

## Local
![image-name](/path/to/image/in/repo)

## External
![image-name](https://www.domain.sthing/path/to/image.png)

Can use html syntax:
<img src = '/path/to/image'>


## Resizing 

<p align="center">
<img src="[/PATH/TO/IMAGE]" width=600>
</p>
